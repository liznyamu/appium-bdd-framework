- [ ] Multiple platform support - iOS and Android
- [ ] Logging framework
- [ ] Reporting framework
- [ ] Capture Screenshots/Videos
- [ ] Different Scrolling Mechanisms
- [ ] Parallel Execution on Android and iOS
